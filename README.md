# cn-space-journal

一个用于归档和发布独立 HTML 页面的仓库。

## 项目简介

本仓库主要用于存放和展示一系列独立的 HTML 页面。这些页面可能涵盖不同主题、不同用途，彼此之间没有强关联，仅作为静态内容归档与发布的载体。

仓库本身不绑定任何特定域名或网站，所有页面均为自包含的 HTML 文件，可直接在浏览器中打开查看。

## 目录说明

```
cn-space-journal/
├── pages/          # 存放独立 HTML 页面
├── assets/         # 公共资源文件（图片、样式、脚本等）
├── README.md       # 本文件
└── ...
```

- `pages/`：每个子文件夹或文件对应一个独立的 HTML 页面，命名清晰，方便查找。
- `assets/`：存放页面可能引用的公共资源，如图片、CSS、JavaScript 等。

## 页面归档说明

- 每个 HTML 页面均为独立、完整的静态文件，不依赖外部服务或动态接口。
- 页面内容可能随时间更新或移除，归档状态以仓库最新提交为准。
- 部分页面可能包含交互功能，但均通过纯前端实现，无需后端支持。
